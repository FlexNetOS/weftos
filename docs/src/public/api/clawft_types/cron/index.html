Cron scheduling types.

Defines the data model for scheduled jobs: CronJob, its CronSchedule, CronPayload, and runtime CronJobState. The CronStore is the top-level container persisted to disk.

All timestamps use DateTime<Utc> for type safety. For backward compatibility, the serde layer accepts both RFC 3339 strings and millisecond-since-epoch integers via custom deserializers.
