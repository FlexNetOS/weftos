Routing and permission configuration types.

Defines the config schema for the TieredRouter (Level 1) and its permission system. All types support both snake_case and camelCase field names in JSON via #[serde(alias)]. Unknown fields are silently ignored for forward compatibility.

When the routing section is absent from config, RoutingConfig::default() produces settings equivalent to the existing StaticRouter (Level 0).
