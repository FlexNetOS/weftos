Mochat channel configuration.

Fields§

§enabled: bool

Whether this channel is enabled.

§base_url: String

Mochat API base URL.

§socket_url: String

WebSocket URL for real-time messaging.

§socket_path: String

Socket.IO path.

§socket_disable_msgpack: bool

Disable msgpack encoding for Socket.IO.

§socket_reconnect_delay_ms: i64

Reconnect delay in milliseconds.

§socket_max_reconnect_delay_ms: i64

Maximum reconnect delay in milliseconds.

§socket_connect_timeout_ms: i64

Connection timeout in milliseconds.

§refresh_interval_ms: i64

Room refresh interval in milliseconds.

§watch_timeout_ms: i64

Watch timeout in milliseconds.

§watch_limit: i64

Maximum messages per watch cycle.

§retry_delay_ms: i64

Retry delay in milliseconds.

§max_retry_attempts: i64

Maximum retry attempts (0 = unlimited).

§claw_token: SecretString

Authentication token.

§agent_user_id: String

Agent user ID for the bot.

§sessions: Vec<String>

Session IDs to monitor.

§panels: Vec<String>

Panel IDs to monitor.

§allow_from: Vec<String>

Allowed user IDs. Empty = allow all.

§mention: MochatMentionConfig

Mention behavior configuration.

§groups: HashMap<String, MochatGroupRule>

Per-group rules keyed by group ID.

§reply_delay_mode: String

Reply delay mode: "off" or "non-mention".

§reply_delay_ms: i64

Reply delay in milliseconds.
