Expand description

Read-side interface shared by all key-value registries.

Implementors expose a uniform way to inspect registry contents without coupling callers to concrete storage (DashMap, HashMap, Vec, etc.).

The Value associated type is the returned value, which may differ from the stored value (e.g., Arc<dyn Trait> vs &T).

§Examples

ⓘ
fn dump_registry(reg: &dyn Registry<Value = String>) {
    for key in reg.list_keys() {
        if let Some(val) = reg.get(&key) {
            println!("{key}: {val}");
        }
    }
}

type Value

The value returned by get.

Use an owned type (Arc<T>, T: Clone) when the registry uses interior mutability and cannot hand out references.
