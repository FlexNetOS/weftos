Post-write verification and hallucination detection.

Free-tier LLMs can hallucinate tool results — claiming files were written when they weren’t. This module provides:

  • Post-write verification: checks filesystem after write/edit tool calls
  • Hallucination score tracking: EMA-smoothed per-session score
  • Complexity boost: inflates routing complexity to push hallucination-prone sessions into higher-tier models via the existing TieredRouter
