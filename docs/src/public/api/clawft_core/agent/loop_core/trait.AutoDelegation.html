Expand description

Trait for pre-LLM automatic delegation routing.

Implementations check whether an inbound message should be routed to a delegation tool (e.g. delegate_task) before the local LLM processes it. This enables rule-based auto-routing for complex tasks that match configured patterns (e.g. “swarm”, “orchestrate”, “deploy”).

If should_delegate returns Some, the agent loop invokes the delegate_task tool directly, bypassing the local LLM entirely. If it returns None, normal LLM processing proceeds.
