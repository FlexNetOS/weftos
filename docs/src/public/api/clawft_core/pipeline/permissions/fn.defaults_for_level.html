pub fn defaults_for_level(level: u8) -> UserPermissions
Expand description

Return built-in defaults for a numeric level.

Levels 0, 1, 2 map to zero_trust, user, admin respectively. Levels >= 3 are treated as admin (highest built-in level) to avoid silently downgrading high-privilege configs to zero_trust.
