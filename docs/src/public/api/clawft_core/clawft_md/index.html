CLAWFT.md loading with import resolution.

Loads workspace-level CLAWFT.md files and resolves @path/to/file import directives. Imports are resolved relative to the CLAWFT.md file’s directory with security constraints (no .. traversal, no absolute paths, bounded recursion depth).

§Import syntax

Lines starting with @ (after optional whitespace) are treated as import directives. The remainder of the line is interpreted as a relative file path:

@prompts/safety.md
@agents/researcher.md

Imports are inlined recursively (up to depth 5). All other lines pass through unchanged.
