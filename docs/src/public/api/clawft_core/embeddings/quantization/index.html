Temperature-based quantization for vector storage (H2.7).

Provides storage-layer quantization that reduces disk/memory usage for less frequently accessed vectors without rebuilding the HNSW index.

§Temperature tiers

TierStorageAccess pattern
HotFull Vec<f32>Frequently accessed
Warmfp16 on diskModerate access
ColdProduct-quantizedRarely accessed

The HNSW index always uses full-precision (f32) vectors. Quantization applies only to the storage/serialization layer. When a warm or cold vector is accessed, it is decompressed to f32 transparently.

Tier transitions are driven by access frequency and recency.

This module is gated behind the rvf feature flag.
