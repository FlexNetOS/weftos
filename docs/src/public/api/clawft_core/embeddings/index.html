Embedding trait definitions and implementations.

Provides the Embedder trait for generating vector embeddings from text, plus a hash_embedder::HashEmbedder that uses SimHash for local, deterministic embeddings with no API calls required.

All types in this module are gated behind the vector-memory feature flag.
