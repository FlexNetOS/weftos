Agent-first architecture: roles, agency, and agent manifests.

In WeftOS, agents ARE the OS. Every feature is delivered by an agent. The kernel boots a root agent which spawns service agents. Each agent has:

  • A role (root, supervisor, service, worker, user)
  • An agency (ability to spawn child agents)
  • A manifest (.agent.toml) describing its capabilities

Agency is hierarchical: the root agent has unlimited agency, supervisors can spawn service/worker agents, services can spawn workers, and workers have no agency (leaf agents).
