Expand description

Execute a calibration run against the provided HNSW index and causal graph.

This inserts config.calibration_ticks synthetic vectors into HNSW, searches each one, creates causal edges linking consecutive ticks, and hashes each vector with BLAKE3 (simulating a Merkle commit). After collecting per-tick timings it computes p50/p95 percentiles, decides the effective tick interval, checks spectral feasibility, and cleans up all synthetic data.
